# Shape Brief — FJ#3: Rebuild theme to conform to DESIGN.md

**Status: CONFIRMED 2026-05-18 (approved as written).** Design-track
per-effort spec (analog of an OpenSpec change record). The craft pass builds
against this; the verify phase (audit re-run + critique + polish) checks
shipped code against it. Inputs: `docs/critique-2026-05-18.md` (21/40),
`docs/audit-2026-05-18.md` (12/20).

## 1. Feature Summary
Bring `frontend/sass/main.scss` and the Hugo templates into conformance with
the DESIGN.md / PRODUCT.md contract via an **aggressive refactor that
preserves the existing token system**. Close every defect in the critique and
audit without altering the fixed contract, except the single
operator-authorized DESIGN.md amendment in §10.

## 2. Primary User Action
An unhurried reader drops into the chronicle and reads a post comfortably,
transported, noticing no seams. The keystone: the small stuff right 100% of
the time.

## 3. Design Direction
Fixed by contract, not re-decided. Committed palette, Nightshade dark ground,
"lived-in spellbook" scene, DESIGN.md's named rules. No per-surface override.
Visual-direction probe skipped: DESIGN.md is the visual spec (no ambiguous
lane) and the harness has no native image generation. Final visual
verification is on the Micro.blog test instance per project process.

## 4. Scope
Production-ready; whole theme (all `layouts/` + `main.scss`); shipped-quality;
polish until it ships. Strategy = **aggressive refactor**: token maps +
accessor functions retained as substrate; violating mixins replaced, missing
systems added, templates restructured where structural. **Conformance +
consolidate**: fix violations and remove the architectural drift that caused
them. Out of scope (FJ#3's pre-filed verify phase): post-build audit re-run,
polish, harden-for-edge-states.

## 5. Layout Strategy
- **De-card the topology.** `.card` remains a DESIGN.md component but stops
  being the universal wrapper. Remove from list page-title
  (`list.html:3-5`) and pagination (`list.html:56`). Article = one quiet
  panel, not a box in a stack. Pagination = unboxed navigation.
- **Real landmarks.** `<nav>` for the Teleportation Circle, `<article>` for
  posts, one `<h1>` per page (site title on home/list; post title on single,
  header site-title demoted off h1 on single). Closes audit dual-h1 / weak
  landmarks.
- **Side scroll: cap + progressive disclosure, no new JS.** `first` caps on
  the unbounded ranges (`baseof.html:82` Allied Mages, `:93` Arcane Tags).
  On single-post pages the article precedes a quieted aside via markup order
  and de-emphasis, not an accordion/toggle. Disclosure by sequence and
  weight so the spell is never "caught happening." CSS/markup only.

## 6. Key States
List: empty / typical / long (pagination). Titled and untitled posts both
render Arcane Tags via one shared partial (fixes `list.html:37-50` /
`index.html:25-38` divergence). Single: with tags / without / with
`arcane-timestamp` (reconcile `single.html` vs `post/single.html`). Archive
(`list.archivehtml.html`). omg.lol fetch success / failure (preserve the
correct `baseof.html:38-68` failure path). Reduced motion: all glow/transition
neutralized. Keyboard focus: every interactive element has a visible on-brand
focus state. Mobile ≤768px: nothing gains the banned stripe (kill
`main.scss:294`); capped lists don't become an endless tail.

## 7. Interaction Model
- Links: teal dotted→solid + teal glow on hover (conformant, unchanged). Add
  distinct `:focus-visible` (solid teal border + glow, differentiated from
  hover) — closes WCAG 2.4.7.
- Chips: rest = transparent 1px border + accent label; hover = border takes
  accent + accent-tinted **contact glow** replacing the `hover-shadow` drop
  shadow (`main.scss:329,344,514`) — fixes Emanation-Over-Elevation.
- Skip-link: visually-hidden-until-focused anchor to `<main>` — closes WCAG
  2.4.1.
- Motion gating: `glow()`/`transition()` wrapped under
  `prefers-reduced-motion: reduce`.
- Random Incantations: **deterministic daily seed** — shuffle keyed to the
  date, stable within a day, fresh daily, excludes the current post. Mechanic
  committed; craft picks a ≤Hugo-0.158 idiomatic seed expression.

## 8. Content Requirements
Diegetic naming preserved verbatim everywhere (load-bearing). Em dash at
`newsletter.html:17` → comma/parenthetical; sweep all templates for
`—`/`--`. `footer.html:2` hardcoded `© 2025` → Hugo `now`-derived year. No
new reader-facing copy; newsletter's micro.blog-mandated mechanical labels
remain the sanctioned backstage exception.

## 9. Recommended References (craft pass, inside the design track)
`harden` (accent-border removal, focus-visible, reduced-motion, skip-link,
landmarks, tokenize SVG ash hue) · `typeset` (70ch measure cap, pagination
off `space(xl)` onto the type ramp) · `colorize` (magenta-chip resolution,
§10) · `adapt` (sub-44px touch targets) · `optimize` (scope `transition:
all`, verify icon/bundle weight) · then `polish`. Run inside the design
track, not ad hoc.

## 10. Resolved Decision (operator-authorized) + Constraint

**Magenta chip AA conflict — RESOLVED: Option B.** `.response-tag` /
`.status-link` magenta `#e31bce` on `rgba(accent,0.15)` over card = 4.41:1,
below the 4.5:1 AA minimum for 0.8rem text. DESIGN.md's "accent at ~15–20%
opacity" and its "WCAG AA non-negotiable" cannot both hold for the magenta
variant.

> **Operator authorization (Contrariwise, 2026-05-18):** craft is explicitly
> authorized to make ONE narrow amendment to the otherwise-frozen DESIGN.md:
> lower the magenta chip background alpha in `main.scss` until the label
> clears 4.5:1, AND record that real alpha in DESIGN.md as a documented
> exception so contract and build stay honest. This is the only sanctioned
> deviation from FJ#3's "Do NOT alter DESIGN.md." Scope is exactly this one
> exception line; no other contract edit is authorized. The gold `.tag`
> (`rgba(secondary,0.2)` = 5.80:1) is unaffected and unchanged.

**Constraint (flag, not a question):** local Hugo is 0.161 but micro.blog
caps at 0.158 (project CLAUDE.md). Craft must not lean on templating
features newer than 0.158 even though the local test build passes. Governs
the daily-seed expression and any template restructuring.
